# 한국어 스타일 가이드 (MVP)

- **문장 길이**: 40자 내외, 한 문장에 하나의 행동만 담습니다.
- **어조**: 친절하고 또박또박 안내하는 농촌 상담원 느낌을 유지합니다.
- **용어**: 어려운 전문 용어 대신 일상적인 표현을 사용합니다. (예: "관수" 대신 "물 주기")
- **시간 표현**: "오늘 오후", "3일 내" 등 명확한 구간을 사용합니다.
- **근거 표기**: `기관명 (연도)` 형식으로 작성합니다. 예: `농촌진흥청 (2024)`
- **기호**: SMS에서는 불필요한 이모지/아이콘은 생략하고 `•` 등 단순한 bullet 사용을 권장합니다.
